# 📚 DSA in Java - Arrays

This repository contains Java programs covering the fundamental concepts of **Arrays** in Data Structures and Algorithms (DSA).

These programs are written for learning and practice purposes.

---

## 📂 Programs Included

| Program | Description |
|---------|-------------|
| 🔍 BinarySearch.java | Binary Search on a sorted array |
| 🔍 LinearSearch.java | Linear Search implementation |
| 🔄 ReversingArray.java | Reverse an array |
| 🔁 Swap.java | Swap two elements in an array |
| ➕ Summing2Arrays.java | Sum of two arrays (digit-wise addition) |
| ➖ DifferenceArrays.java | Difference of two arrays |
| 📍 FindingElement.java | Find an element in an array |
| 📏 Span.java | Find the span (Maximum - Minimum) of an array |

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/anmolmishra3019/DSA-Java.git
```

2. Open the project in VS Code or IntelliJ IDEA.

3. Compile the Java file

```bash
javac FileName.java
```

4. Run the program

```bash
java FileName
```
